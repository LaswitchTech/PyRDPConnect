## Description
`PyRDPConnect` is a remote desktop client based on FreeRDP 2. It is a front-end for thin-clients.

## Features
  - Support Microsoft RDP Connections (Windows Terminal Server and Direct Connection)

## Planned
  - Support for openVPN
